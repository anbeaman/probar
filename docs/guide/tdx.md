# 通达信 `pb.tdx`

**规划中(v0.2)**。通达信走**二进制 TCP 协议**(默认 7709 标准 / 7727 扩展),不复用 HTTP 层。
定位是 A 股**行情底座**:批量实时、历史分钟、**历史逐笔**是它相对东财的强项;复权由 probar 用除权除息数据自算。

当前命名空间已声明接口面,调用会抛 `NotImplementedError` 并注明计划版本:

| 接口 | 说明 |
|---|---|
| `quote` / `quotes` | 实时快照(批量强) |
| `kline(freq=日/周/月/分钟, adjust=...)` | K 线(复权自算) |
| `intraday` / `intraday_hist` | 当日 / 历史分时 |
| `ticks` / `ticks_hist` | 当日 / 历史逐笔(分笔成交明细,**非** L2 逐笔委托) |
| `xdxr` | 除权除息(用于复权) |
| `securities` | 全市场代码表 |
| `block` | 本地板块(依赖本地 TDX 数据) |
| `finance_info` | 基础财务 |

!!! note "命名空间里没有什么"
    通达信协议**没有**资金流 / 龙虎榜 / 北向数据域,因此 `pb.tdx` 里**不存在** `fund_flow` / `lhb` / `hsgt`
    —— 访问会得到 `AttributeError`,而不是运行时"不支持"。这些请用 `pb.dc`。

!!! warning "服务器池"
    v0.2 会内置服务器池并做**业务探针**(拉固定股票校验返回长度/时间/价格区间),而非仅 ping —— 因为"能连 ≠ 数据完整"。
