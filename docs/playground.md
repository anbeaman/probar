# 接口测试台

一个本地可视化网页:列出全部接口,选接口、填参数、运行,查看**输入 / 输出**(表格 + 来源 + 耗时),
错误也清晰回显。不用写代码就能试。

```bash
pip install "probar[playground]"
python -m probar.playground          # 打开 http://127.0.0.1:8787
# 自定义端口:python -m probar.playground --port 9000
```

用法:

1. 左侧选数据源/接口(**● 已实现 / ○ 未实现**,支持搜索)。
2. 中间自动生成参数表单(必填项标 `*`,带默认值与类型提示;`list` 类型逗号分隔)。
3. 点**运行** → 右侧表格展示返回(列 + 行)、来源 `provenance`、耗时;出错显示异常类型与信息。
4. 右上角"能力矩阵"查看三源能力总览。

!!! warning
    仅本地开发/测试用,会**真实联网取数**;请勿对外暴露该端口。
