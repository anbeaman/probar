# 能力矩阵

代码里随时可查:

```python
import probar as pb
pb.capabilities()      # 返回 DataFrame:行=能力,列=dc/tdx/ths
```

图例:**✅ 强**(可做主实现) · **🔸 部分/需计算** · **⚠️ 反爬脆**(best-effort) · **❌ 无**

| 能力 | 东财 dc | 通达信 tdx | 同花顺 ths |
|---|:--:|:--:|:--:|
| 实时快照(单/批量/全市场) | ✅ | ✅ | ⚠️ |
| 五档盘口(仅 L1) | ✅ | ✅ | ⚠️ |
| 当日分时 | ✅ | ✅ | ⚠️ |
| 历史分时 | 🔸 | ✅ | ⚠️ |
| 当日逐笔 | 🔸 | ✅ | ⚠️ |
| 历史逐笔 | ❌ | 🔸 非L2·唯一可取 | ❌ |
| K线 日/周/月 | ✅ | ✅ | ⚠️ |
| K线 分钟 | 🔸 1分历史浅 | ✅ | ⚠️ |
| 前/后复权 | ✅ 一参直给 | 🔸 用 xdxr 自算 | ⚠️ |
| 资金流(主力/超大单/板块) | ✅ | ❌ | ⚠️ |
| 龙虎榜 | ✅ | ❌ | ⚠️ |

| 证券代码表 | ✅ | ✅ | 🔸 |

| 多市场(港美/期货/基金/转债) | ✅ 最广 | 🔸 扩展行情 | ⚠️ |

!!! note "重要口径"
    - 三个免费源给的都是 **Level-1 五档**,没有 L2 十档/委托队列/逐笔委托。
    - 通达信**没有**资金流/龙虎榜/北向(协议无此数据域);其"历史逐笔"是分笔成交明细,非交易所 L2。
    - 东财北向**盘中实时**已停披露,仅额度 + 盘后/EOD。
    - 该矩阵是**数据源能力参考**,不是方法清单:各命名空间暴露其中已实现/计划实现的方法子集,实际可调用以 `dir(pb.dc)` 为准。
